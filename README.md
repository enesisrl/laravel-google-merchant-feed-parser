# Laravel Google Merchant Feed Parser

1. Aggiungi a `composer.json`
    ```
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/enesisrl/laravel-google-merchant-feed-parser"
        }
    ]
    ```
2. Esegui
    ```
    composer require enesisrl/laravel-google-merchant-feed-parser
    ```
